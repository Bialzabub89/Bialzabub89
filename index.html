Html Elements

Tag Name Explanation
<head> </head> Head This is where your meta data is stored.
<body> </body> Body This defines the documents body.
<div> </div> Div Defines a section within the html page. Can also nest other div tags within.
<h1> to </h6> Headings Html Headings ranging from largest h1 to smallest h6.
<a> </a> Anchor Defines a Hyperlink
<b> </b> Bold Defines bold text. This should ideally be done within CSS however.
<button> </button> Button Creates a clickable button on the webpage.
<figure> </figure> Figure Specifies self contained content.
<figcaption> </figcaption> Figcaption Defines a caption for a <figure> element.
<table> </table> Table Defines a table.
<thead> </thead> Table Header Groups the header content in a table. Nested within Table Row.
<tbody> </tbody> Table Body Groups the body Content within a table
<tr> </tr> Table Row Defines a row in a table.
<th> </th> Header Cell Defines a header cell in a table. Nested within Table Row and Table Header.
<tfoot> </tfoot> Table Footer Groups the footer content in a table.

CSS Selectors

Example Name Explanation
.example {} Class selector This will allow you to target any html with a class defined within your html. <p class="example">
#example {} ID selector This will allow you to target any html with an id defined. <p id="example">
p{} Element selector This will allow you to target all html paragraph elementsand can target all html elements. <p>
*{} Universal selector This will select all HTML elements on the page.
